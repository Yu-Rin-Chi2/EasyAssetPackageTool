# 機能要件

## MoSCoW 分類

### Must Have (実装済み)

| ID | 機能 | 状態 |
|----|------|------|
| F-01 | EditorWindow での GUI 操作 | 済 |
| F-02 | Include パスの追加・削除 | 済 |
| F-03 | Exclude パスの追加・削除 | 済 |
| F-04 | ワイルドカード除外パターン (`*`, `**/`) | 済 |
| F-05 | パッケージ名の `{version}` 置換 | 済 |
| F-06 | 依存関係の包含/除外切り替え | 済 |
| F-07 | 設定の ScriptableObject 永続化 | 済 |
| F-08 | .unitypackage ビルド実行 | 済 |
| F-09 | 出力フォルダを開く | 済 |
| F-10 | 含まれるアセット一覧のプレビュー | 済 |
| F-11 | 除外ファイル数の表示 | 済 |
| F-12 | デフォルト設定へのリセット | 済 |

### Should Have (未実装)

| ID | 機能 | 備考 |
|----|------|------|
| F-13 | プリセット設定の切り替え | 複数の設定を保存・切り替え |

### Could Have (未実装)

| ID | 機能 | 備考 |
|----|------|------|
| F-14 | ビルド履歴の記録 | 過去のビルド情報をログとして保持 |
| F-15 | コマンドライン (CLI) ビルド対応 | CI/CD 連携用 |

### Won't Have (スコープ外)

- Asset Store 向けの検証・アップロード機能
- パッケージの署名・暗号化
